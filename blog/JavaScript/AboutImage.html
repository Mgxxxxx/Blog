# 上传图片

一种简单的上传图片的方式是使用HTML5的 "<input type="file" />" ,获取到用户选择的图片后，可以把文件添加到FormData对象上，然后设置http请求头的"Content-type"字段为"mutipart/form-data"，最后将表单提交给服务器。

const file = document.querySelector(".input");
file.addEventListener("input", (e) => {
  const image = e.target.files[0];
  const data = new FormData();
  data.append("image", image);
  const xhr = new XMLHttpRequest();
  xhr.open("post", "/uploadimage", true);
  xhr.onreadystatechange = () => {
    if (xhr.readyState == 4) console.log(xhr.responseText);
  };
  xhr.send();
})

# 解析二进制数据为图片

这是一个较简单的过程，首先要把二进制数据转换为Blob对象。可以在请求头的Header中添加"responseType"字段为"blob"，这样服务器传来的数据就会自动抓换为Blob对象。接着使用window.URL.createObjectURL(blob)方法，返回值为一个url，可以设置img的src为这个url，即可显示改图片。这个url会存在内存中，若不使用，应使用window.URL.revokeObjectURL(url)来释放。

# 论坛项目中有关帖子中用户头像的一些优化

在论坛项目中，是先从后端获取二进制数据，然后使用window.URL.createObjectUrl(blob)获得图片的url，该url会留在内存中，故，我们可以将该url保存到localStorage。在帖子中，同一个用户就不需再次向后端请求头像，也不用再生成一次url，直接使用localStorage中对应的url即可。 由于浏览器会自己缓存，所以并不会像后端发送多次请求。

# 图片展示

之前展示图片是从后端接受数据后转化为blob对象，然后再转化为图片的url来展示，其实可以直接把获取到的数据直接放在img标签的src属性上即可展示。
