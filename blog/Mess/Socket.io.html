# 自制微型聊天室学习socket.io

该聊天室使用的是Ts+Vue3+Sass+Socket.io来完成的。具体完成的功能如下:

  • 所有用户在聊天室Room中聊天
  • 所有用户在聊天室Room中可以知道谁上线和谁下线
  • 显示各个用户的在线状态
  • 用户之间可以进行私聊
  • 有新消息会有小红点提示

# 创建socket实例

首先我们要下载socket.io对应的包：

yarn add socket.io socket.io-client
1

一开始我只下载了socket.io，但是在客户端使用ts找不到创建实例的接口，最后查资料得还需下载socket.io-client包。服务端和客户端创建实例代码如下：

//server
const app = require("http").createServer(function (req, res) {});
const io = require("socket.io")(app, { cors: true });
app.listen(15656)

//client
import io, { Socket } from "socket.io-client";
class Connect {
  private client: Socket;
  constructor(url: string) {
    //创建套接字
    this.client = io(url, { autoConnect: false });
    this.client.connect();
  }
}

以上代码中，服务端设置了cors属性，为的是解决跨域问题。在客户端设置了autoConnect属性为false。socket.io默认会为我们自动连接，这里我们自己手动连接。这里提一点，服务端创建的时候若不使用require("socket.io").of("nsp")这种形式创建的话，socket.io会自动把我们分配到默认命名空间(即namespace)"/"。指定了的话，则为对应的nsp。这个案例中，我把默认命名空间作为聊天室。

//server
socket.on("private msg", function (msg, to) {

私聊消息事件设置为private msg。这里我是为两个用户之间创建了一个room，这个room的id我并没有设置为他们自己的socket的id，我自己创建了一个关系表，两两用户之间有一个唯一的id，room的id就为该id，这样做是考虑到以后可能会有多端登录。若使用socket自己的id，则私聊消息只会有一个端收到，其他收不到。服务端收到一条私聊消息之后，就会找到这两个用户对应的房间的id，然后to(id),找到对应房间，发送消息。

# 在聊天室内知晓他人上下线消息,显示用户的在线状态

//server
io.on("connection", function (socket) {
  socket.broadcast.emit("user connected", user);
  socket.on("disconnect", () => {
    // console.log(user.name + " leave.");
    socket.broadcast.emit("user disconnected", user);
  });
}

//client
this.client.on("user connected", (user: User) => {
  //有用户上线则推送消息，并设置用户状态为在线
  }
});
this.client.on("user disconnected", (user: User) => {
  //有用户上线则推送消息，并设置用户状态为离线
});

# 有新消息，展示小红点

这里我们可以在接收到新消息的时候进行处理，当接收到新消息的时候，我设置了对应的用户的数据结构中的hasNewMsg为true，当用户读了消息的时候就设置为false。在进行渲染的时候，我就可以根据这个值来决定是否显示小红点。

# 总结socket.io

这里就简单的总结一下socket.io的一些常用方法：

io.of("nsp")    //设置命名空间
io.emit("event")    //向所有用户发送消息
io.of("nsg").emit("event")    //向命名空间内的所有用户发送消息

io.on("connect",function(){})   //监听连接事件

socket.emit("event")    //向客户端发送消息
socket.broadcast.emit("event")    //向所有客户端，除了发送者发送消息
socket.join(id)   //加入一个房间
socket.to(roomid).emit("event")   //向某个房间发送消息

socket.on("connect",function(){})   //监听连接事件
socket.on("disconnect",function(){})   //监听断开连接事件
