# "script setup"中使用"inheritAttr"

在vue3中我们可以这样使用inheritAttr：

export default defineComponent({ 
  inheritAttrs: false,
  setup () {
     // ... 
  } 
})

若在script setup中我们并不会使用defineComponent()，但是我们可以在template上的<template></template>定义这个属性:

<template inherit-attrs="false">
  <div>
    <button v-bind="$attrs">Ciao!</button>
  </div>
</template>
